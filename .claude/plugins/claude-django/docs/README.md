# Django Plugin Documentation

This directory contains reference documentation for Django development best practices, coding standards, and large project patterns.

## Files

| File | Description |
|------|-------------|
| `standards.md` | Coding standards, naming conventions, and structural guidelines for Django projects |
| `best-practices.md` | Best practices for security, performance, testing, deployment, and more |
| `large-project-patterns.md` | Organizational patterns for large projects with per-class file structure |

## Usage

These reference documents are automatically included in the context when using Django plugin commands, ensuring consistent coding standards across your projects.

## Quick Reference

**large-project-patterns.md:**
